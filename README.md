# Hybrid Augmented Intelligence in the Vehicle Routing Problem (VRP)

This repository contains the datasets and preprocessing scripts supporting the paper:

**“From Models and Algorithms to Decision Makers:  
A Review of Hybrid Augmented Intelligence in the Vehicle Routing Problem.”**

---

## 🎯 Objective

This study investigates how human expertise, metaheuristics, machine learning, fuzzy systems, and decision-support frameworks are being combined in VRP research (2015–2025).

The central question is:

> Are current approaches truly implementing *Augmented Intelligence* (Human + AI),  
> or mostly hybridizing algorithms (AI + AI)?

The review focuses on the intersection between:

- Metaheuristics and hyper-heuristics  
- Machine learning–guided optimization  
- Fuzzy systems  
- Multi-criteria decision-making (MCDM)  
- Preference modeling  
- Human-in-the-loop systems  
- Decision-support architectures
